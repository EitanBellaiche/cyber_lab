Start the attacker page server:
python3 -m http.server 5500

Vulnerable demo:
1. Run: sudo ./zookld --vulnerable
2. Victim logs in at: http://localhost:8080/zoobar/index.cgi/
3. Open attacker page: http://localhost:5500/csrf-transfer.html?mode=vulnerable
4. Expected result: transfer succeeds

Fixed demo:
1. Run: sudo ./zookld
2. Victim logs in at: https://localhost:8443/zoobar/index.cgi/
3. Open attacker page: http://localhost:5500/csrf-transfer.html?mode=fixed
4. Expected result: request is blocked by CSRF protection
